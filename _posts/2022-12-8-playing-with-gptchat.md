After seeing some mention of what GPT Chat could do I decided that it would be a good idea to have a play with it myself, I had heard that it was able to generate code and I wanted to see how usable this feature would be.

Starting with a simple description of what I was looking for the AI produced something that I didn't expect actual code it wasn't the best quality but asking for further refinements improved the quality it formatted the python code I asked for too pep8 but it was still inefficent. I was trying to get it to write a program that would sort photos by date and put them into dated files, what I found was the more specific my request the better the result in the end I didn't end up with working code but it was very close to being a working solution.

Looking at the code it produced it's easy to see defficencys but this wasn't what the program was designed to do, no matter how hard I tried I couldn't make it add efficencies that I had tested and knew were more efficent. While I dont think us software engineers have anything to worry about it is intresting technology and one day could become more useful.
